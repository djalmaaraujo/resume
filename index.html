Worth to mention

Some projects that makes me love what I do everyday.

FaceGroup

github.com/djalmaaraujo/facegroup

Facegroup is a new perspective of how groups should looks like. I have too many groups to read and get updates, so the main goal of this project is creating a new way of reading groups, using the same idea as twitter lists. You can choose which groups are related to a specific subject, then group all into a single timeline. So if you have 5 gruops of Javascript/Node, you can read a single timeline with all posts. Also, there are some good features like: Comments, Favorite posts, Cross-plataform view, etc. This a working in progress project.

StoriaJS

github.com/djalmaaraujo/storiajs

Storia is simple and small library which allows you to create single page applications using the HTML5 History API. It gives you the power to use only html-markup to create a fully navigation website using pushState feature. It's very simple to use and you don't even need a back-end to start creating your website with different URL's for different content pages.

customelements.io

github.com/webcomponents/customelements.io

CustomElements.io is where community-members show their awesome Web Components. I'm an active contributor for this project. I believe that the web is about to change and Web components is a good path to follow. I also maintain the fetch-service repository, which is the crawler for all custom elements. We fetch bower, npm and github projects into a single api. fetch-service repository
