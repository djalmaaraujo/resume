FaceGroup

github.com/djalmaaraujo/facegroup

Facegroup is a new perspective of how groups should look like. I have too many groups to read and get updates from, so the main goal of this project is to create a new way of reading groups, using the same idea as twitter lists. You can choose which groups are related to a specific subject, then group all into a single timeline. For example, if you have 5 groups of Javascript/Node, you can read a single timeline with all posts. Also, there are some good features like: Comments, Favorite posts, Cross-plataform view, etc. This a working in progress project.
